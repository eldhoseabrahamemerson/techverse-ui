# ⟨ Techverse /⟩ — Technology Learning Platform

A responsive frontend template for a technology learning platform, built with pure HTML and CSS. This project was created to practice and demonstrate core CSS layout techniques including Grid, Flexbox, and Float — all in one real-looking website.

## 🎯 Purpose

Techverse is a static UI concept for a technology learning platform. It serves as a visual mockup of where beginners might find tutorials, projects, and resources for web development. This is a frontend concept project created to demonstrate modern HTML and CSS layout techniques through a realistic technology learning platform.
---

## 🛠️ Built With

- **HTML5** — Semantic structure (`header`, `nav`, `main`, `aside`, `section`, `footer`)
- **CSS3** — Layouts, transitions, hover effects, and responsive design
- No frameworks. No JavaScript. Just the fundamentals.

---

## 📐 CSS Concepts Covered

This project is a hands-on demo of several key CSS layout techniques:

| Concept | Where It's Used |
|---|---|
| **CSS Grid** | Content + Sidebar layout, Cards section |
| **Flexbox** | Navigation bar |
| **Float** | Image wrapping inside article |
| **Box Model** | Padding, margin, border-radius throughout |
| **Transitions** | Hover effects on nav links and sidebar items |
| **Media Queries** | Responsive layout for mobile screens |
| **Pseudo-elements** | `::after` clearfix for float container |

## 📱 Responsive Design

The layout adjusts for smaller screens:

- Navigation stacks **vertically** on mobile
- Content + Sidebar switches to a **single column**
- Cards stack **one per row**
- Floated image centers and goes **full-width**
